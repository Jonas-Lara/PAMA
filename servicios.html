Nuestros servicios

Desde Integración de maquinaria industrial a mantenimiento especializado como procesos láser, Contamos con un amplio catálogo de servicios.

Servicios

Mantenimiento a equipo láser, Industrias NORM Septiembre, 2019

Mantenimiento especializado

Descripción del proyecto

Mantenimiento a equipo láser, nuestro trabajo consistió en ajustar los cordones de soldadura láser, modificando el programa de trayectoría del robot y ajustando parametros de soldadura, garantizando el trabajo con la aprobación del departamento de calidad.

Integración de engargoladora, Industrias NORM Diciembre, 2019

Integración industrial

Descripción del proyecto

En este proyecto se modificó un proceso de engargolado desmontando la isla y reintegrando la engargoladora para proceso manual; modificamos el programa de PLC y el programa del bus de campo, para satisfacer las necesidades de producción y ahorro de energías auxiliares de nuestro cliente.

Desmontaje total de cabinas láser, Industrias NORM Marzo, 2020

Montaje y desmontaje de maquinaria

Descripción del proyecto

Se realizó el desmontaje de una cabina láser en su totalidad, siendo extremadamente cuidadosos para desmontar los dispositivos de ensamble, los brazos roboticos y los cabezales láser, así poder reintegrarlos modificando el proceso de producción.

Desarrollo de nuestro sitio web September, 2020

Desarrollo de software

Descripción del proyecto

Desarrollamos nuestra página web, para mostrar a futuros clientes que contamos con ingeniería para desarrollo de software, para este proyecto utilizamos tecnologías como JavaScript, jquery y firebase, además de aplicar tecnologías basadas en núbe para el análisis de datos.
